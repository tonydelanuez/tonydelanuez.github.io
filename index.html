PROJECTS

ARDUINO FITBIT CLONE



For a class project I created a Fitbit clone with step tracking, temperature detection, a sleep tracking mode, and live graphing of the accelerometer data through a Java GUI. This was also a lesson in protocols as I created a custom communication protocol to send messages and data from the Arduino to the Java program used for calculations and display output. I used a Sparkfun board DEV-12757 and the SparkFun Triple Axis Accelerometer (MMA8452Q) for the tracker. In order to effectively track step data and filter out noise from the accelerometer, I computed a rolling average of the input data and created both peak-detecting and threshold-matching algorithms to detect what was actually a full step. In the picture above, the green dots are the peaks at where a step was detected. Once 45 seconds passes, the x-axis adjusts and starts at 45(this increments every 45 seconds). The pedometer can be reset, and when the device is set to sleep tracking mode it will continue to count time spent under a certain threshold of movement (I allowed a certain amount of movement to account for tossing and turning in sleep). The device also displays a dynamic step rate, calculated using the time since last reset of the pedometer. Modes are indicated both in hardware by two LEDs and in software via the GUI. I did not create housing for the device, this was all set up on a breadboard. Code not publicly available for academic integrity.

ECHOCARDIOGRAM DESIGN
Washington University in St. Louis



For the final project of a microelectronics class, my colleague Prakrit Shrestha and I designed an amplifying circuit that would take the signal from electrodes placed at different spots on the human body, and amplify it to a level easily viewable on a computer. The project was a lesson in circuit theory, common-mode voltage filtering, grounding, and proper usage of Wilson's central terminal. We were required to implement the above ideas and more in order to create a design with an acceptable amount of noise and a very clear, stable signal.
